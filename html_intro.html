HTML Intro & Basics

What Is HTML?

It stands for HyperText Markup Language.

It is used to create web pages.

Hypertext refers to connect websites internally as-well-as externally created by other peoples also.

It is the most basic building block of web.

Created by W3C (World Wide Web Consortium).

HTML Versions

Currently HTML Living Standard (HTML5) is running which gets regular updates.

Last numbered version is HTML5.2 (2017).

Major difference between HTML4 & 5 is:

  1. Semantic tags like header, footer, article etc.
  2. Graphical Tag canvas.
  3. Media Tags like audio, video.
  4. Form input types added like email, date, range, color, datalist, progress etc.
