2. Introduction to body tags

  • Block Elements

These elements occupies whole width of the web page.
All Content written inside these tags always comes into new line.

  • Heading Tags like <h1> to <h6>
  • <p> Tag used for paragraph
  • <br/> & <hr/> used for breaking row and providing horizantal row
  • <pre> for pre-formatted text like 
                        Hello ,     How Are You?
                    I am Fine   
                        Thank You!
  • <q> for inline quotations and <blockqoute> for long quotations.
  • <cite> is used to provide author name of blog, quotes, poems, songs etc.
