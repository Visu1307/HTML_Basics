51. <iframe> allowfullscreen It specifies whether the content is allowed to go in full screen or not.
loading Indicates when the browser should load the iframe:
eager : Immediately loads when page gets loaded.

It takes path absolute or relative.
referrerpolicy It allows you to define how much referrer to send when following link.
Possible Values : no-referrer, no-referrer-when-downgrade, origin, origin-when-cross-origin, same-origin, strict-origin, strict-origin-when-cross-origin, unsafe-url

52. <img> src It allows you defining absoulte or relative path to show the image.
alt It allows you defining alternate text if browser fails to load the image.
Can also be used for Screen Readers.
loading It indicates browser when to load the image :
eager: It directly loads the image when page gets loaded.
lazy: It only loads image when the image is at some calcuated distance from view port.
decoding When to decode the image possible values
sync: It decodes the image along the DOM creation.
async: It decodes the image after DOM creation.
referrerpolicy It allows you to define how much referrer to send when following link.
Possible Values : no-referrer, no-referrer-when-downgrade, origin, origin-when-cross-origin, same-origin, strict-origin, strict-origin-when-cross-origin, unsafe-url
fetchpriority It allows you defining relative fetch priority over other images.
high: It fetches image more earlier than other images.
It fetches image late relative to other images.
auto Browser decide on its own.
srcset It allows you defining multiple images along with view width and image source.
sizes Only be used if srcset is present. Defines max and min width when to load which image from srcset images.
usemap Allows using map and area tags to define URLs upon image.
